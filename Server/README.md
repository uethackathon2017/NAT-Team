# UET Hackathon 2017
## NAT Team - CrossWorld

### Server Repo

##### Mô tả
- Server NodeJS phục vụ cho [client iOS](../Client-iOS)

##### Xây dựng trên
- NodeJS (v6.10.0 LTS)
- Express (v4.15.2)
- Socket.io (v1.7.3)
- Mysql Server (v5.7.17)

##### Sử dụng
1. Cài đặt các thư viện yêu cầu: `npm install`
2. Chạy server: `node index.js`
